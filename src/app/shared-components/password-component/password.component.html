• Passwords must be at least ten (10) characters but no more than one hundred (100) characters
  • Passwords must contain at least 1 uppercase character (A-Z)
  • Passwords must contain at least 1 lowercase character (a-z)
  • Passwords must contain at least 1 digit (0-9)
  • Password must contain at least 1 special character such as #$@!%&*?
  • Passwords may not contain user's first name, last name or user ID
